About JASMIN and CEDA

For this hackathon, you will be using the NERC JASMIN high performance computer system. JASMIN is operated by the Science and Technology Facilities Council on behalf of the Natural Environment Research Council and provides powerful computational resources for environmental data analysis.

JASMIN allows direct access to data held in the NERC CEDA data archive. CEDA (Centre for Environmental Data Analysis) is NERC's primary data centre for atmospheric and earth observation research, housing petabytes of environmental data including weather observations, climate models, satellite data, and more.

These accounts will give you full access to the JASMIN notebook server and CEDA data archive.

Step-by-Step Connection Instructions

1. Complete the online registration form

Navigate to the online hackathon registration form using your web browser to obtain a JASMIN account:

Scan the QR code above with your phone to access the registration form, or click on this Registration Link.

You will see the JASMIN login page where you can sign in with your assigned hackathon credentials.

2. Access the JASMIN Notebook Server

Navigate to the JASMIN notebook server using your web browser:

URL: http://notebooks.jasmin.ac.uk/

You will see the JASMIN login page where you can sign in with your assigned hackathon credentials.

3. Log In with Your Hackathon Account

Use the username and password provided to you at the beginning of the hackathon session. These credentials will give you access to:

  • The JASMIN notebook server environment
  • Direct access to the CEDA data archive
  • Pre-configured Python environments with all necessary libraries
  • Computational resources for data analysis

4. Access and Clone the Hackathon Notebooks

Once logged in, you will need to access and clone the notebooks prepared for the session. There are four notebooks to access:

  • ECMWF Data Challenges - Global weather and climate reanalysis data
  • MIDAS Weather Station Data - Station-based weather observations
  • HadUK-Grid Climate Data - Gridded UK climate data
  • Soils and Land Cover - Spatial environmental datasets for Bedfordshire

It is suggested you first create a new folder on your JASMIN account to hold the notebooks, e.g. 'hackathon' and any other files you will create during the hackathon. Next, go into this folder and clone the notebooks. This is achieved via the 'git' menu item in the Jupyter notebook interface, or the git clone icon. Enter in the git repo to clone, e.g. https://github.com/NERCDigitalGathering2025/NERCDigitalGathering_Hackathon_2025. The files are then copied down in to your account for you to work through. Further instructions for accessing and cloning these notebooks will be provided during the hackathon session.

5. Start Your Analysis

With the notebooks cloned and ready, you can begin exploring the environmental datasets and working through the hackathon challenges. The JASMIN environment provides:

  • High-performance computing resources
  • Direct access to CEDA data without downloading
  • Pre-installed Python libraries for data analysis
  • Collaborative workspace for team projects

üí° Getting Your Own JASMIN Account

While you'll have access through the hackathon accounts, you can also apply for your own JASMIN account if you'd like to continue using these resources after the event:

  • JASMIN Account: https://jasmin.ac.uk
  • CEDA Account: https://archive.ceda.ac.uk

These accounts are free for UK academic and research users.

Troubleshooting

If you encounter any issues connecting to JASMIN or accessing the notebooks:

  • Ensure you're using the correct URL: http://notebooks.jasmin.ac.uk/
  • Check that your internet connection is stable
  • Try refreshing the page or clearing your browser cache
  • Contact the hackathon organisers for assistance with account credentials
